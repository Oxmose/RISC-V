
# RISC-V 
Simple and non optimized implementation of a RISC-V processor. This RISC-V pipeline is based on the DLX architecture.

Implementation: RISC-V32I

## Current state:
* CPU        : Done
* CPU Bypass : Done 
* CPU Stall  : Done
* CPU Stomp  : Done
* CPU CSRs   : In progress
* Caches     : TODO
